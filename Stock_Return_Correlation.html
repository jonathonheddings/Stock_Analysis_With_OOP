How Correlated Are Two Stocks Returns Over Time?¶

Using pandas, seaborn, and a stock module I developed, I graph the correlation between returns with different windows between two stocks

import stock_class as stock
import pandas as pd
import numpy as np
import seaborn as sns
sns.set()

stock1 = "MSFT"
stock2 = "SPY"

test1 = stock.Stock(stock1)
test2 = stock.Stock(stock2)
test1, test2

daily_returns1 = test1.returns['D']['Returns']
daily_returns2 = test2.returns['D']['Returns']
